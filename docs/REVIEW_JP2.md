**対応内容:**
- コマンドインジェクション対策: PowerShellコマンドのBase64エンコーディングと`-EncodedCommand`の使用
- パストラバーサル対策: パス検証と`Path.resolve()`を使用した安全なパス処理
- 入力サニタイズ: シングルクォートのエスケープ (`'` → `''`)
- サブプロセスセキュリティ: `shell=False`の明示的設定

**コード例:**
```python
# Before (脆弱)
ps_cmd = f"... {user_input} ..."
result = subprocess.run(full_cmd, capture_output=True, timeout=10)

# After (安全)
ps_command = self._build_ps_command(req)
full_cmd = [ps_exe] + self.ps_args + ["-EncodedCommand", self._encode_command(ps_command)]
result = subprocess.run(full_cmd, capture_output=True, timeout=self.timeout, shell=False)
```
